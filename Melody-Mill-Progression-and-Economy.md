# Melody Mill - Progression and Economy

## Purpose
This document defines the best path for pacing, prestige, meta progression, and economy tuning in **Melody Mill**.

## Core Recommendation
Build progression in this order:

1. Tune `Cat Mill` from first click to first prestige.
2. Add the `Harmony Points` loop and a small global `Harmony Tree`.
3. Add a second Mill only after the first prestige cycle feels good.

Do not tune multiple Mills at once. A clicker game lives or dies on pacing, so the first working loop should be:

- click
- buy upgrade
- hear the song improve
- reach a stage milestone
- prestige
- feel stronger next run

## Pacing Targets
These are the recommended targets for a first playable build.

### First Session
- First meaningful upgrade: `20-40 seconds`
- First automation source: `2-4 minutes`
- First clear visual stage change: `4-6 minutes`
- First major musical payoff: `5-8 minutes`
- First prestige opportunity: `20-30 minutes`

### Repeat Sessions
- Second prestige: `10-15 minutes`
- Mature prestige loop: `8-12 minutes`
- First new Mill unlock: `90-120 minutes` total playtime

### Music Improvement Cadence
- Early game: the song should noticeably improve every `1-2 minutes`
- Mid game: every `3-5 minutes`
- Late game: every `5-10 minutes`, plus occasional rare payoff unlocks

## Cat Mill Target Run
This is the recommended structure for the launch Mill.

### Minutes 0-2
- Player clicks yarn manually
- Buys first low-cost unit quickly
- Hears the first repeating note layer

### Minutes 2-5
- Unlocks a second unit type
- Gains the first real automation feeling
- Song adds a second lane or stronger rhythmic pulse

### Minutes 5-10
- First multiplier or support upgrade appears
- Yarn ball visibly changes stage
- Music feels fuller and more "produced"

### Minutes 10-20
- Most early upgrades are active
- Player begins choosing between cost efficiency and faster growth
- Third and fourth lanes become more prominent

### Minutes 20-30
- Player reaches the first meaningful prestige milestone
- Game clearly explains the `Spin Into Harmony` reset
- First prestige should feel exciting, not punishing

## Economy Structure
Use two currencies:

- `Local resource`: each Mill has its own main resource
- `Harmony Points`: a global prestige currency shared across all Mills

Examples:
- Cat Mill -> `Yarn`
- Robot Mill -> `Energy`
- Spooky Mill -> `Echo`

## Upgrade Categories
Each Mill should use the same upgrade shape, even if names differ.

### Common Units
- frequent purchases
- low initial cost
- cost scaling around `1.15-1.18`
- provide most of the early momentum

### Support Units
- medium cost
- cost scaling around `1.18-1.22`
- improve multipliers, timing, or other units

### Rare Units
- expensive
- cost scaling around `1.25-1.35`
- bring strong musical identity and large production jumps

### Capstone Units
- late run purchases
- very high cost
- should feel like "the song is complete now"

## Resource Balance Rules
- Clicking should matter for the first `8-12 minutes` of a run
- Automation should provide about `70-80%` of income by the midgame
- Support upgrades should feel powerful, but never replace the excitement of buying a new unit type
- Every `5-10` purchases should produce a visible or audible payoff

## Main Object Progression
The main object should work like a stage ladder, not a health sponge.

### Stage Rules
- each new stage should be visually larger and more satisfying
- early stage requirements should scale by roughly `8x-12x`
- later stage requirements can scale by `15x-25x`
- each stage should unlock either a new upgrade, a new music layer, or a stronger multiplier

For Cat Mill, this means the yarn ball should not just get "bigger." It should feel more important and more musical.

## Prestige Model
Prestige should be local per Mill, but feed a global meta layer.

### Recommendation
- each Mill can be reset independently
- prestige becomes available after a major stage milestone
- prestige grants `Harmony Points`
- Harmony Points never reset

### Cat Mill Fantasy
In Cat Mill, the player can `Spin Into Harmony` by converting a fully developed yarn run into Harmony Points.

## Harmony Point Pacing
The first few prestiges should come quickly enough to teach the loop.

### Recommended Rewards
- First prestige: `3-5 Harmony`
- Second prestige: `5-8 Harmony`
- Third prestige: `8-12 Harmony`

This is enough to make the tree feel active without exploding the economy.

## Harmony Tree Spending Targets
Keep the early tree generous.

### Suggested Costs
- Small stat node: `1-3 Harmony`
- Useful quality-of-life node: `3-5 Harmony`
- Strong automation or music node: `5-8 Harmony`
- Major unlock: `10-15 Harmony`

## New Mill Unlocking
Do not hide the first new Mill too deep in the tree.

### Best Path
- first prestige unlocks the concept of Harmony
- first few Harmony purchases make the player feel stronger
- first new Mill becomes available after `3-5 prestiges`
- target cost for the first new Mill: around `25 Harmony`

The first new Mill should feel earned, but not so late that players assume the game only has one world.

## Harmony Tree Structure
The global tree should stay small in version 1.

### Recommended Branches
- `Rhythm`: timing bonuses, auto-hit speed, beat-based buffs
- `Harmony`: extra layers, richer chords, stronger music density
- `Craft`: cheaper upgrades, better offline rewards, stronger starts
- `Discovery`: new Mills, theme slots, customization capacity

### Version 1 Size
- target `15-25 total nodes`
- avoid trap choices
- allow respec or cheap rerouting if branching matters

## Offline Progression
Offline gains should exist, but stay controlled.

### Recommendation
- base offline reward: `20-30%` of online production
- cap at `8 hours` in version 1
- improve through Craft nodes later

This helps casual players without making active play feel pointless.

## UI Clarity for Progression
Players should always understand which layer they are interacting with.

### Separate Screens
- `Mill View`: local resource, upgrades, stage progress
- `Harmony Screen`: prestige explanation and global currency
- `Harmony Tree`: permanent upgrades
- `Mill Select`: unlocked worlds

Do not mix Theme Pack controls into the prestige or world-unlock screens.

## Tuning Risks
- If first prestige takes longer than `30 minutes`, many players will bounce.
- If first prestige takes less than `15 minutes`, the run may feel too shallow.
- If the first new Mill arrives before the prestige loop is satisfying, expansion will hide a weak core.
- If music upgrades are too sparse, the game will feel like a normal clicker with a soundtrack instead of a music game.

## Best Path Summary
1. Make `Cat Mill` fun for the first `30 minutes`.
2. Add the first prestige and `Harmony Points`.
3. Tune the first `10-15` Harmony spends.
4. Unlock the second Mill only after the first prestige loop feels reliable.
